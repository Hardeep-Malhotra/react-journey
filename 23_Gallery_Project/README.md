# 📸 Photo Gallery Pro

A modern, responsive image gallery built with **React**, **Tailwind CSS**, and **Axios**. This project demonstrates how to handle API data, implement pagination, and provide a smooth user experience with loading states.

---

## ✨ Features

- **Dynamic Data Fetching:** Fetches high-quality images from the Lorem Picsum API.
- **Modern UI:** Built with a dark theme using Tailwind CSS, featuring glassmorphism and smooth hover animations.
- **Pagination:** Efficiently browses through hundreds of images using "Next" and "Previous" controls.
- **Loading States:** Includes a custom CSS spinner to improve UX during data fetching.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop screens.
- **Auto-Scroll:** Automatically scrolls to the top of the page when switching pages for a seamless feel.

## 🚀 Getting Started

Follow these steps to run the project locally:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/photo-gallery-pro.git
   ```

2. **Navigate to the project folder:**

```bash
cd photo-gallery-pro
```

3. **Install dependencies**

```bash
npm install
```

4. **Start the development server**

```bash

npm run dev
```

## **📖 What I Learned**

During this project, I mastered several key concepts of modern web development:

| Concept                   | Description                                                                                 |
| :------------------------ | :------------------------------------------------------------------------------------------ |
| **State Management**      | Using `useState` to track image data, page numbers, and loading status.                     |
| **Side Effects**          | Using `useEffect` to trigger API calls whenever the page state changes.                     |
| **Conditional Rendering** | Showing/hiding the spinner and content based on the `isLoading` state.                      |
| **API Integration**       | Handling asynchronous code with `async/await` and managing errors with `try/catch/finally`. |
